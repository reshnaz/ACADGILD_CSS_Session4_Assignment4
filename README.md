# ACADGILD_CSS_Session4_Assignment4
Animate a bicycle image.

Problem Statement

--Create a Div box and insert an image as bicycle inside the Div box and perform the
following tasks :

-Drive the image with animation timing duration to 5s.

-Animate the image using animation-timing-function.

-Add animation iteration property to make the animation infinite.

-Animate the image from margin left 0px to left 500px. Use @keyframes.
